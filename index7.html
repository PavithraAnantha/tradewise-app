TradeWise API Documentation
Trade Services API v1 · 99 endpoints · tradewise-api.vercel.app
🔍
Party Screening 1 endpoint
POST
screen-party
Screens a party against known party lists and determines if there's a match entity.
HS Classification 14 endpoints
POST
determine-hs
Determines the best HS identification and alternative classifications for a commodity based on product name and description.
GET
validate-hs
Determines the validity of an HS number.
GET
search-tariff
Returns the HS numbers from a tariff based on a keyword search.
GET
map-hs
Maps an HS number of a source country to equivalent HS number(s) of a target country.
GET
get-child-hs
Returns the children HS lists for a given HS code.
GET
get-hs-details
Returns the HS details information (+).
GET
get-hs-chapters
Returns the HS chapters for an import or export country.
GET
get-hs-subtitles
Returns the HS sections entries for a given HS code.
GET
get-hs-uoms
Returns a list of UOMs for an HS number.
GET
get-path-to-hs
For a given HS code, returns the HS nodes from the chapter level to the final HS number.
POST
determine-product-hs-misclassification
Determines if a submitted HS classification may be incorrect based on product name and description.
POST
determine-supplemental-hs
Determines the supplemental HS number that may be applicable for a specified HS number.
POST
validate-supplemental-as-primary-hs
Determines if supplemental HS numbers have been incorrectly used in place of a primary HS number.
Export Controls (ECN) 14 endpoints
POST
determine-ecn
If trading (exd) and (isd) a commodity, determines if it is controlled for dual or military use based on product name and set.
POST
determine-ecn-export-controls
Determines the ECN-based export controls that may apply for an export country.
POST
determine-ecn-export-control-exceptions
Determines exceptions that may apply for a specified ECN export control number.
GET
explain-ecn-export-control-applicability
Explains why an ECN export control number may apply.
GET
validate-ecn
Determines the validity of an ECN number.
GET
search-ecn
Returns the ECN numbers from a keyword search.
GET
get-ecn-categories
Returns the categories for an exporting country's export control numbers.
GET
get-ecn-exceptions
Returns a list of ECN-based export control exceptions for an export country.
GET
get-ecn-export-controls
Returns a list of ECN-based export controls for an export country.
Embargoes & Sanctions 3 endpoints
GET
determine-embargoes-sanctions
Determines the embargoes and sanctions that may apply for an import or export country.
GET
explain-embargo-sanction-applicability
Explains why an embargo or sanction may apply in a given trade scenario.
GET
get-embargoes-sanctions-details
Retrieve information for one or more embargoes or sanctions (description, filing info…).
Import Duties & Costs 20 endpoints
GET
determine-duties
Determines applicable import duties (general (mfn/gst), national programs (mtfn/sct), for your HS code.
GET
determine-duty-exemptions
Determines the duty exemptions or reductions that may apply.
POST
calculate-duty
Returns the total applicable duty for imported goods based on HS code, country, quota info, and more.
GET
get-SPIs
Returns a list of SPIs for a country.
